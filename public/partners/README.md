# Partner Logos

This folder contains partner/sponsor logos displayed at the bottom center of the screen.

## Supported Formats

- PNG (recommended for logos with transparency)
- JPG/JPEG
- GIF
- SVG
- WebP

## Recommended Specifications

- **Height**: 100-200px (auto-scaled to 50px)
- **Format**: PNG with transparent background
- **File Size**: Under 500KB per logo
- **Aspect Ratio**: Any, automatically adjusted

## Installation

1. Upload sponsor logos to this folder
2. Restart server
3. Logos appear automatically on all viewer pages

## Example Filenames

```
public/partners/
├── sponsor1.png
├── sponsor2.png
├── partner_logo.svg
└── main_sponsor.png
```

## Notes

- Logos are automatically sorted alphabetically
- All logos are displayed horizontally in a row
- The bar scrolls automatically with many logos
- Logos without background look best
