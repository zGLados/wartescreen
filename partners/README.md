# Partner-Logos

Dieser Ordner enthält die Partner-Logos, die unten in der Mitte des Bildschirms angezeigt werden.

## Unterstützte Formate

- PNG (empfohlen für Logos mit Transparenz)
- JPG/JPEG
- GIF
- SVG
- WebP

## Empfohlene Spezifikationen

- **Höhe**: 100-200px (wird automatisch auf 50px skaliert)
- **Format**: PNG mit transparentem Hintergrund
- **Dateigröße**: Unter 500KB pro Logo
- **Seitenverhältnis**: Beliebig, wird automatisch angepasst

## Installation

1. Sponsor-Logos in diesen Ordner hochladen (via FTP oder direkt)
2. Server neu starten oder warten bis Auto-Reload
3. Logos erscheinen automatisch auf allen Viewer-Seiten

## Beispiel-Dateinamen

```
public/sponsors/
├── sponsor1.png
├── sponsor2.png
├── partner_logo.svg
└── hauptsponsor.png
```

## Hinweise

- Logos werden automatisch alphabetisch sortiert
- Alle Logos werden horizontal in einer Reihe angezeigt
- Die Leiste scrollt bei vielen Logos automatisch
- Logos ohne Hintergrund sehen am besten aus
